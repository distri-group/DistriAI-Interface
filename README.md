# DistriAI-Interface

## Features

- Buy computing power easily and quickly;
- Earning from sharing your computing power;
- Security ensured by blockchain technology.

## Install

node:^21.1.0

```
yarn
```

## Run

```
yarn start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser
